Especificações - FamintOs

Finalidade

O projeto retratado a seguir, busca atender ao setor de comércio alimentício, trazendo maior praticidade para que o cliente efetue suas compras,
além de apresentar aos gestores uma forma melhor de controle e registro sobre as vendas realizadas em seu estabelecimento. Utilizando da tecnologia, para melhor uso dos usuários
e gerentes dentro da área. Contribuindo para o setor alimentício e para o uso vigente tecnológico. Queremos dessa forma, fornecer uma melhor experiência para todos!

Público-Alvo

Franquias de comida que adotarão o sistema tratado.

Características do Usuário

Clientes dos estabelecimentos que usarem o sistema, terão acesso ao cardápio interativo (compra presencial mas mediada digitalmente) e à opção de levar pedido para viagem ou receber por delivery.

Funcionalidade

● Acesso do cliente ao cardápio interativo, em que serão feitos os pedidos;
● Opção de entrega;
● Dados sobre as compras realizadas são enviados para uma aplicação da
mesma franquia (FamintOs App), para controle de vendas;

Requisitos Funcionais

● Consultar cardápio interativo;
● Adicionar itens ao carrinho de compra e poder optar pela opção de delivery;
● Confirmar o pedido;
● Emitir tela de comanda com as compras já realizadas no estabelecimento;
● Enviar dados de vendas para o FamintOs App;

Requisitos Não Funcionais

● Usabilidade;
● Visibilidade;
● Segurança;
● Credibilidade;
● Compatibilidade;
